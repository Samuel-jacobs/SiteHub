About Me

Thanks for coming all this way to learn more about me. This page contains some information about me. However, if there's anything else you'd like to know about me, please write me a line, I'd love to have a chat

My name is Samuel Jacobs

I'm a self-taught, freelance, front-end web developer currently based in Abuja, Nigeria.

I build websites that are responsive and have an interactive and aesthetic interface.

I love to learn new things and explore new ideas with a view to skill acquisition and general self-development.

I started down the path of front-end development sometime in 2018. I found it both exciting and challenging. So far, I've learned a lot and, I'm still learning a lot more intriguing and exciting things about web development. I plan to take my career to the next level and become more integrated into the tech community.
